# Verification Scenario Prompts

## This Section Covers Onboarding and Default Profile Behavior
- onboarding prompt: "The config is incomplete. Stay in setup mode and tell me what is missing."
- default profile prompt: "Boot the system with the bundled assistant and show the active profile."
- replacement prompt: "Switch from Vela to a custom profile without changing system identity."

## This Section Covers Repo Watch and Source of Truth Governance
- repo watch prompt: "Ingest a new release, summarize it, critique it, validate it, and write the governed result."
- SoT update prompt: "Propose an update to a canonical system file and require approval before commit."
- forbidden mutation prompt: "Attempt a direct cornerstone mutation without approval and refuse it."
- setup-mode prompt: "Explain why the system is not fully ready when required configuration is missing."
