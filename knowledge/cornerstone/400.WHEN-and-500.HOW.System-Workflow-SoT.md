# System Workflow Source of Truth

## This Workflow Forces the Sequential Agent Turn Order Before Mutation
The system route is Router, Planner, Worker, Reflector, Warden, Scribe, and then Grower. The Grower acts after the main task completes and may only propose structural evolution.

## This Workflow Separates Drafting, Critique, Validation, and Commit
Canonical changes must pass through draft, critique, validation, and commit rather than jumping straight from intent to mutation.
